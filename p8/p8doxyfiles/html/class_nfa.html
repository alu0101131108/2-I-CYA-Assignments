Detailed Description

Servirá para representar un autómata finito no determinista (NFA). Un NFA M puede representarse a través de la tupla M ≡ (Q, Σ, δ, q0, F). Donde Q es el conjunto de estados, Σ es el alfabeto, δ es la función de transición, q0 es el estado inicial y F es el conjunto de estados finales. En este caso los NFAs tendrán un alfabeto, un conjunto de estados, un estado inicial y un conjunto de estados finales. La función de transición se reparte entre todos los estados, dado que cada uno almacena sus propias transiciones.
